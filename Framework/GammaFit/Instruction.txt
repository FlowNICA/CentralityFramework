Для реконструкции необходимо задавать значение прицельного параметра, есть два варианта
1) значение известно и вы задаете его заранее(пусть например оно равно 607.084), тогда необходимо в макро GammaFit2.C в строке 17 поставить bool GetSigma = false;
и задать значение сечения в строке 18
Float_t sigma = 607.084;
теперь необходимо запустить макро и на вход подать файл с гистограммой и нижней границей фита
 root GammaFit2.C++'("./refMult_UrQMD_11gev_500k.root","hRefMultSTAR","./out.root",10,"hBvsRefMult")'
 где первая переменная входной рут файл -"./refMult_UrQMD_11gev_500k.root"
 вторая название гистограммы с множественностью - "hRefMultSTAR"
 третья выходной файл "./out.root"
 4-ая нижняя граница фита -10(Nch_{min})
 5-ая название двумерной гистограммы, что бы можно было сравнить результаты ,"hBvsRefMult"
 если вам не нужно сравнивать с результатами из модели то нужно закоментить 573 строку, и раскоментить 574
 2) значение сечения не известно, тогда можно просто в строке 17 поставить bool GetSigma = true;
 и тогда программа сама посчитает сечение из двумерной гистограмки
 запускать точно так же
 root GammaFit2.C++'("./refMult_UrQMD_11gev_500k.root","hRefMultSTAR","./out.root",10,"hBvsRefMult")'

 (примечание)
 1) начальные параметры фита хорошо работаю для большего диапозона энергий и тяжелых систем, но иногда если множественность слишком маленькая или же система легкая, то нужно немного подкорректировать начальные значения,но это бывает необходимо крайне редко
 2) границы классов по центральности задаються с помощью массива range_cent (стр 26), так же нужно не забывать менять const int CentClass на 24 строке в соотвествии с размером

 После фита для получения функции центральности необходимо использовать макро printFinal.C

root printFinal.C'("./out.root","out.C")'

первая переменная - входной файл с результатами фита
вторая переменная- выходной файл в формате c++
